# Raylib Halo Clone Prototype

Phase 1 bootstrap for a small Halo-inspired first-person shooter prototype built with C# and Raylib-cs.

## Requirements

- Windows development machine
- .NET 8 SDK or newer
- NuGet package: `Raylib-cs` version `8.0.0`

## Build

```powershell
dotnet restore
dotnet build -c Debug
```

## Run

```powershell
dotnet run -c Debug
```

## Controls

- Mouse: look around
- W/A/S/D: move
- Left Shift: sprint
- Space: jump
- Esc or window close: quit
